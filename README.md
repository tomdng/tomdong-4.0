## Getting Started

To get a local copy up and running follow these simple steps.

- Get NodeJS and Yarn

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/github_username/repo_name.git
   ```
2. Install packages
   ```sh
   yarn install
   ```
3. Run in dev mode
   ```sh
   yarn dev
   ```
4. Visit `localhost:3000`!
